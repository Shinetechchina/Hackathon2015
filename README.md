#Cloud Life 智慧家庭

##项目描述
使用树莓派构建智慧家庭控制系统，实现远程手动、自动现控制家庭电器.

##功能描述

###智能盒子
* 上传设备的状态
* 控制设备的开关

###服务器
* 与智能盒子建立连接
* 提供API供终端调用

###终端
* 设置智能盒上挂载的设备
* 显示、控制设备的状态
* 定时控制设备的状态
* 通过GPS 信息，控制设备的状态  （当手机远离房间的时候， 自动关闭室内的灯)
